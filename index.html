Snapshot Capture from Webcam

Build a real-time webcam application that shows a live video feed and lets the user do two things:

  • Quit the video by pressing the q key.
  • Take a snapshot of the current frame by pressing the s key.

This simulates how most camera apps work ‚Äî giving you the live preview, and capturing the image only when you're ready.

Step 1: Set up project folder

  • Create a folder named webcam_capture
  • Inside this folder, create a Python file named snapshot.py

Step 2: Import the OpenCV library

  • Start the file by writing import cv2

Step 3: Connect to the webcam
Use cv2.VideoCapture(0) and store the result in a variable named cap.

Step 4: Start an infinite loop
Write a while True: loop to keep reading frames from the webcam.

Step 5: Read the frame
Inside the loop, write ret, frame = cap.read() to capture the frame and check if it was successful.

Step 6: Exit if the frame is not read
Immediately after reading, check if not ret: and if True, write break.

Step 7: Display the frame in a window
Use cv2.imshow() to show the frame in a window titled "My Video".

Step 8: Read the key press
Store the result of cv2.waitKey(1) in a variable called key.

Step 9: Check if the user has pressed ‚Äòq‚Äô

  • Use if key == ord('q'):
  • If this condition is True, use break to exit the loop.

Step 10: Check if the user has pressed ‚Äòs‚Äô

  • Use elif key == ord('s'):
  • If this condition is True, use cv2.imwrite('snapshot.png', frame) to save the frame, and print a message saying ‚ÄúSnapshot saved as 'snapshot.png'‚Äù.

Step 11: Release the webcam
After the loop, write cap.release() to close the camera connection.

Step 12: Close all OpenCV windows
Write cv2.destroyAllWindows() to shut down all OpenCV-created windows.
